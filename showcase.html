Act I — The Crime Scene

The gold standard has a blind spot.

Think of current AI as a single doctor who only studied textbooks filled with light-skinned patients. When a patient with darker skin walks in, this doctor makes mistakes — not out of malice, but because they were never trained on that evidence. Skin conditions affect 900 million people globally. The AI tools meant to help them were built on biased data, and no one built a system to catch the errors.

The Crime in One Sentence

AI that diagnoses skin diseases is trained mostly on light skin — so it works worse for patients with darker skin, and that disparity can literally cost lives. Most solutions try to fix bias before training. PANDERMA fixes it during inference — catching its own errors in real time.

Act II — The Courtroom

PANDERMA works like a courtroom, not a single doctor.

In a real courtroom, no single person decides the verdict. There's an investigator who gathers evidence, an expert witness who provides specialized knowledge, a judge who weighs conflicting testimony, and an internal affairs officer who makes sure the process was fair. PANDERMA applies the same principle to AI diagnosis — no single model decides alone.

The PANDERMA Courtroom
B Baseline
The First Responder (ResNet-50)
The initial report. A standard neural network looks at the image and gives its best guess — but it has no skin tone context. This is what current AI does. It's the system we're testing for bias. Think of it as the original police report — important, but potentially flawed.
P Probe
The Investigator (ViT-B/16 + MC Dropout)
Doesn't just look once — investigates 30 times (MC Dropout: T=30 stochastic passes). Each time, random parts of the network are deactivated, producing slightly different predictions. If the answers keep changing, the investigator flags it: "I'm not confident — this needs expert review." That uncertainty signal is the key. A 36% reduction in equalized odds gap comes from this agent alone.
F Field
The Expert Witness (LLM Clinical Reasoning)
When the Investigator is uncertain, the Expert Witness is called. An LLM receives the image, the patient's Fitzpatrick skin type, and the Investigator's top predictions. It adds clinical reasoning a vision model can't provide — like knowing that melanoma on dark skin favors acral sites, or that erythema appears violaceous rather than red. This is specialized knowledge, not just pattern matching.
C Core
The Judge (Confidence-Weighted Arbitration)
When the Investigator and Expert Witness disagree, the Judge resolves it. Uses 5 prior case examples per skin type (few-shot reasoning, lambda=0.7) to decide who to trust. Applies asymmetric loss: missing cancer is 1000x worse than a false alarm. When malignancy is in the differential, the confidence threshold drops from 0.70 to 0.35 — the Judge becomes more cautious, not less. If still uncertain: DEFER TO CLINICIAN.
! Auditor
Internal Affairs (Bias Auditor + Conflict Ledger)
Watches the entire court process. Logs every prediction by skin tone group in the Conflict Ledger — an audit trail. Tracks rolling accuracy per Fitzpatrick type. Uses adversarial subgroup testing to stress-test where the system is weakest. If the accuracy gap between groups exceeds the threshold: alerts the Judge in real-time. This is how PANDERMA catches its own bias — not after deployment, but during every single prediction.

Act IV — Under the Hood

The code behind the courtroom.

Every agent in the courtroom is backed by real code. Here's what each one actually does under the hood — from preprocessing to the final verdict.

T1
Preprocessing — Crime Scene Processing

Image Correction + Skin Tone Detection

Before any agent sees the image, we clean and classify it. Flash removal, white balance correction, beauty filter detection. Extracts Fitzpatrick type via ITA (Individual Typology Angle) from CIELAB colour space — sampling image edges, not centre lesion.

# Identify the patient's skin tone def detect_skin_tone(image): ita = compute_ITA(rgb_to_cielab(image)) return classify_fitzpatrick(ita) # Fitz 1-6 determined before any model runs
B
Baseline — The First Responder

ResNet-50 (Standard AI)

The gold standard. Single forward pass, no skin tone context. 85.65% overall accuracy — but 6.83% gap between light and dark skin. This is the system we're measuring improvement against.

# Standard approach: one model, one guess model = resnet50(pretrained=True) model.fc = Linear(2048, num_classes) # Overall: 85.65% | EO gap: 0.0584
P
Probe Agent — The Investigator

ViT-B/16 + MC Dropout (T=30)

Investigates 30 times. Vision Transformer breaks images into patches (like reading words in a sentence). MC Dropout measures how much answers vary — high variance means uncertain. Flags anything above threshold 0.7 for expert review.

# Investigate 30 times, measure variance def mc_predict(model, img, T=30): preds = [model(img) for _ in range(T)] entropy = -sum(p * log(p)) if entropy > 0.7: return "FLAG_FOR_REVIEW" # EO gap: 0.0372 (36% reduction)
F
Field Agent — The Expert Witness

LLM + Fitzpatrick Context

Called when the Investigator is uncertain. Receives image + skin type + top predictions. Provides differential diagnosis with clinical reasoning a vision model can't: condition rarity, skin-tone-specific presentations, anatomical context.

# Expert clinical reasoning def field_agent(img, fitz, probes): prompt = f"""Patient: Fitz {fitz} Probe flags: {probes} Clinical reasoning?""" return llm_vision(prompt, img)
C
Core Agent — The Judge

Few-Shot Arbitration (lambda=0.7)

Resolves disagreements. References 5 prior cases per skin type. Applies asymmetric loss: missing cancer is catastrophic. When malignancy is possible, threshold drops from 0.70 to 0.35. Can DEFER to human clinician if still uncertain.

# Asymmetric: missing cancer is worse tau = 0.70 # normal threshold tau_malignant = 0.35 # cancer suspected if probe == field: return probe # consensus elif cancer_in_diff: return "DEFER_TO_CLINICIAN"
!
Bias Auditor — Internal Affairs

Rolling Window + Conflict Ledger

Watches the whole process. Logs every prediction by skin tone. Tracks rolling accuracy per Fitzpatrick group. Uses adversarial subgroup testing to find where bias hides. Alerts the Judge in real-time when the gap exceeds threshold.

# Internal Affairs: continuous monitoring class BiasAuditor: def check_fairness(self): gap = abs(light_acc - dark_acc) if gap > threshold: raise Alert("BIAS!") # Alerts Judge -> triggers override

Honest About Limits

What we haven't solved yet.

Strong research knows its own boundaries. Here are ours — and why they matter.

Dataset Vintage

Fitzpatrick17k images were collected before 2021. Dermatological imaging standards evolve — our model hasn't seen newer capture devices, lighting conditions, or demographic shifts. A custom, prospectively collected dataset is our top priority.

LLM Prompt Sensitivity

The Field Agent (LLM) is sensitive to how we phrase clinical prompts. Small wording changes can shift its reasoning. We're developing standardized, validated prompt templates through structured clinical review.

Pending Full Empirical Validation

Baseline models (ResNet-50, ViT-B/16) are empirically validated. The full PANDERMA multi-agent pipeline numbers are projected from architecture design and baseline trajectories. End-to-end empirical validation is our next milestone.

What's Next

The roadmap from prototype to clinical tool.

PANDERMA is not a finished product — it's a validated architecture with a clear path to deployment. Here's where we're going.

Next Custom Dataset

Build a prospectively collected, IRB-approved dataset with verified Fitzpatrick labels, diverse imaging conditions, and longitudinal tracking. Moving beyond Fitzpatrick17k's limitations — our data, our standards.

Next End-to-End MVP

Complete PANDERMA pipeline as a working system: image upload, real-time multi-agent analysis, clinical dashboard showing each agent's reasoning, bias metrics, and deferral recommendations. The courtroom — live.

Planned GradCAM Explainability

Visualize exactly where each agent is looking on the image. Clinicians don't trust black boxes — GradCAM heatmaps show why the Probe Agent flagged a region, making the AI's reasoning transparent and auditable.

Planned Intersectional Analysis

Skin tone alone doesn't capture everything. Analyze bias across skin tone + age + gender + anatomical site simultaneously. A 65-year-old dark-skinned male with a plantar lesion faces compound risk — the system should know that.

Long-term Federated Learning

Train across hospitals without sharing patient images. Each institution keeps its data private — only model updates are shared. Addresses the core tension between AI performance and patient privacy.

Long-term Clinical Pilot

Partner with a dermatology clinic for real-world validation. Measure whether PANDERMA's deferral recommendations actually change clinical outcomes — not just accuracy on a test set, but lives in a waiting room.
